# Linux & Bash Automation for DevOps Operations

## Description
Created Bash scripts for log cleanup, disk monitoring, and service checks. Improves operational efficiency through basic automation.

## Technologies
- Linux
- Bash Scripting

## Usage
- Run `./log_cleanup.sh` for logs.
- Run `./disk_monitor.sh` for disk.
- Run `./service_check.sh` for services.
- Run `./devops_automation.sh` for all.
